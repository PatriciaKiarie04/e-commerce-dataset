# E-Commerce Data Analysis

## Overview
Exploratory analysis of an online retail dataset containing 1000 rows 
and 13 columns covering customer behavior and sales trends.

## What I did
- Loaded and explored the dataset
- Identified and handled missing values in Gender and Review Score columns
- Cleaned data using mode and mean imputation

## Tools Used
- Python
- Pandas
- Google Colab
